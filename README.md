## Installation

- Install [`pyenv`](https://github.com/pyenv/pyenv).
- Install [`pyenv-virtualenv`](https://github.com/pyenv/pyenv-virtualenv).

```
pyenv install 3.8.1
pyenv virtualenv analyzer
pyenv activate analyzer
pip install -r requirements.txt
```
